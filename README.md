# NutriTrack – Calorie Tracker Web App

NutriTrack is a simple **web app for tracking calories and macros**.  
It helps users easily log their meals, monitor daily goals, and see weekly progress.

[Live Demo on Vercel](https://yummy-ai-scanner.vercel.app)

## Features
- Quick and easy meal logging
- Daily calorie goal tracking
- Overview of protein, carbs, and fats
- Weekly chart for progress visualization
- Serverless function for premium access / payments (dummy test)

## How It Works
1. Add meals using the "Add Food" button
2. See daily totals in the Calorie Ring
3. Track weekly intake in the chart
4. Click the "€10 Access" button to test premium access (dummy checkout)

## Tech Stack
- **Frontend:** React / Next.js  
- **Backend:** Vercel Serverless Functions  
- **UI Components:** TailwindCSS + Lucide Icons  
- **AI Integration:** Optional for calorie estimates or automation
